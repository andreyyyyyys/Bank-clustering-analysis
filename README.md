# Анализ банковских транзакций: кластеризация клиентов

Кластерный анализ клиентов банка для выявления групп риска и разработки стратегий удержания.

---

## О проекте

**Цель:** сегментировать клиентов по транзакционному поведению, выявить группы риска и предложить адресные меры удержания.

### Основные этапы
1. Предобработка данных
2. Исследовательский анализ
3. Создание агрегированных признаков 
4. Стандартизация и кластеризация 
5. Оценка устойчивости кластеров
6. Интерпретация и разработка стратегий

---

## Данные

- **Источник:** `bank_transactions_data_2.csv` 
- **Транзакции:** 2 513
- **Клиенты:** 500
- **Период:** 2023-01-02 — 2024-01-01

*Данные можно скачать по [ссылке](https://www.kaggle.com/datasets/valakhorasani/bank-transaction-dataset-for-fraud-detection)*

---

## Ключевые результаты

- Выделено **4 кластера** клиентов с разным уровнем лояльности
- Разработаны **адресные стратегии удержания** для каждого кластера
